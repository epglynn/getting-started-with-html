World Wide Web is a collection of HTML documents.

- HTML is hypertext markup language
- text content (what you see)
- markup (what it looks like)

- hyperlinks glue them together
- browser displays files found on the web
- internet world's largest computer network
- HTTP main protocol of the web
- servers computers that host files that make up the web

Tags, Attributes and Elements

- tag can have multiple attributes. Tags that are not closed are called "void" tags and contain no content/text.
- element is everything that falls inside of a opening and closing tags.
